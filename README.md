Этот код на Python позволяет объединять 2-а списка в 3-ёх разных ситуациях
Запуск программы
1. Установить Python
2. Скачать файл `list-merge-project`
3. Запустить в терминале
После того как пользователь запустил программу, он вводит два списка в неограниченном количестве индексов через пробел, а затем выбирает режим слияния списков.
Режимы объединение:
1. Обычный - Не сортирует числа, просто объединяет числа пользователя которые он введёт вне зависимости от его порядка.
2. Без повторов - Проверяет введёные числа на наличие повторяющихся чисел, если они есть, программа удаляет лишние числа и оставляет только одно число, затем объединяет.
3. Сортировка - Сортирует числа введёные пользователем а затем объединяет в порядке возрастания.
